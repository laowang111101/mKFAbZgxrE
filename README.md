# 学生成绩管理系统简介 python809

## 项目概述

学生成绩管理系统是一款基于Python语言开发的管理系统，以Django作为后端框架，Vue.js作为前端技术。该系统致力于实现对学生成绩的高效管理，提供了多种功能模块，满足教师、学生等不同角色的需求。

## 技术栈

- 后端：Python 3.7 + Django
- 前端：Vue.js
- 数据库：MySQL
- 开发工具：PyCharm

## 功能模块

- 首页：展示系统概况
- 个人中心：管理个人信息
- 教师管理：处理教师相关信息
- 学生管理：管理学生资料
- 公告信息管理：发布和管理公告信息
- 课程类型管理：维护课程类型数据
- 课程信息管理：管理课程详情
- 选课信息管理：记录学生的选课信息
- 成绩信息管理：处理学生成绩数据

## 系统角色

- 系统管理员：负责全局配置与系统维护
- 教师：发布成绩，管理课程
- 学生：查询成绩，进行选课

## 运行环境

- Python 3.7 或以上版本
- Django 框架
- Vue.js 前端框架
- MySQL 数据库

## 部署步骤

1. 安装Python 3.7及其以上版本
2. 安装Django框架
3. 安装MySQL数据库
4. 克隆前端Vue项目代码，并部署至Web服务器
5. 运行后端Django项目
6. 数据库迁移与初始化
